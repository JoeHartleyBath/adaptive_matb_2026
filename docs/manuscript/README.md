# Manuscript workspace (thin scaffold)

This folder is a deliberately minimal workspace to keep *canonical context* close to the project while we start the study.

## What this is

- A place to store stable context that helps future writing assistance (humans and tools).
- A lightweight home for a small set of canonical references and their BibTeX entries.

## What this is not

- Not a venue decision.
- Not an outline.
- Not a draft.
- Not a results folder.

## Boundaries

- Venue, framing, and manuscript structure are intentionally **not fixed**.
- PDFs do **not** belong here (keep this repo free of large binaries and copyrighted artifacts).
